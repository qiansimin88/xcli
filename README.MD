# @xc/xox
简小的下载各种业务类型模板的脚手架下载工具（仅仅是模板，运行时构建功能依附于各个模板运行时脚手架）

## 安装
```js
yarn global add @xc/xox  (Npm源切换到http://npm.xinc818.com)
```

## API
在你需要建立项目的目录
```
xox init [yourProjectName👋👋👋]
```
